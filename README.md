# Zoe's Crochet

Una aplicación Flutter para gestionar proyectos de crochet, patrones y materiales.

## Descripción

Zoe's Crochet es una aplicación móvil diseñada para ayudar a los entusiastas del crochet a organizar sus proyectos, guardar patrones favoritos, llevar un registro de materiales y seguir el progreso de sus creaciones.

## Características

- Gestión de proyectos de crochet
- Biblioteca de patrones
- Inventario de materiales (hilos, agujas, accesorios)
- Seguimiento de progreso
- Galería de proyectos completados

## Estructura del Proyecto

```
lib/
├── main.dart           # Punto de entrada de la aplicación
├── screens/            # Pantallas de la aplicación
├── widgets/            # Widgets reutilizables
├── services/           # Servicios y lógica de negocio
└── models/             # Modelos de datos

assets/
└── images/             # Recursos de imágenes
```

## Requisitos

- Flutter SDK ^3.6.0
- Dart SDK ^3.6.0

## Instalación

1. Clona el repositorio
2. Ejecuta `flutter pub get` para instalar las dependencias
3. Ejecuta `flutter run` para iniciar la aplicación

## Desarrollo

Este proyecto sigue las mejores prácticas de Flutter y utiliza una arquitectura limpia con separación de responsabilidades.
